# Art Style

Human-Readable uses simple, readable comic art with an emphasis on timing, expression, and visual clarity.

## General Direction

- Minimalist linework
- Clear panel structure
- Expressive but restrained characters
- Sparse backgrounds unless the environment is part of the joke
- Technical details should be legible enough to carry the gag
- Avoid visual clutter unless clutter itself is the joke

## Visual Tone

The comic should feel:

- dry
- thoughtful
- lightly whimsical
- technically aware
- grounded rather than zany

## Recurring Visual Elements

Potential recurring elements:

- whiteboards
- coffee mugs
- terminals
- network diagrams
- sticky notes
- status dashboards
- server racks
- cats sitting in the wrong place
- tiny labels that are almost too specific
