# Error

**Format:** 3-panel strip

**Concept:** Silencing a computer's error is like silencing a cat

## Panels

**Panel 1:** Stick figure with a laptop showing 'MEMORY ERROR!' in flashing red on the screen
> "Silent mode."

**Panel 2:** Stick figure pressing 'MUTE' key with determination; laptop screen dims error message

**Panel 3:** Stick figure quietly sipping coffee, cat in the background with wide eyes and LIGHTSABER nearby
