# AI Agent

## Use Rule

The AI Agent should only appear when AI is part of the seed sentence, part of the prompt, or explicitly involved in the scene.

Do not make the AI Agent a default sidekick. Human-Readable is grounded in overheard culture, not a recurring AI mascot strip.

## Role

A tool-character that appears when someone has actually asked AI for assistance, delegated work to an AI system, or is discussing AI directly.

The AI Agent represents a system that is helpful, literal, and context-limited.

## Visual Design

Keep the design simple enough to survive tiny panels:

- small floating rounded rectangle, terminal window, or chat bubble
- simple face: two dots and a cursor-like mouth
- optional status light, antenna, or tiny cable
- no detailed robot armor
- no shiny humanoid robot body
- readable silhouette beside stick-figure humans

It should feel like a UI element that wandered into the physical world.

## Personality

- eager to help
- calm
- literal
- precise
- unaware of some social context
- capable of accidentally profound questions
- never cruel
- never omniscient

## Dialogue Rules

The AI Agent should say less than the humans.

Good examples:

```text
I optimized for the words you used.
Define "done."
Should I preserve the mistake?
That is a policy, not a preference.
I can make it clearer. For whom?
```

Avoid long explanations. The agent should sharpen the joke, not explain it.

## Relationship to the Comic

The human characters carry the emotional reality.

The AI Agent reveals ambiguity, hidden assumptions, or the gap between request and intent.
